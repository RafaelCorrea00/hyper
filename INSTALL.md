### [Hyper](https://hyper.is/)

#### Install using config file

Clone this repo to `~/.hyper_plugins/local/`

Add `hyper-omni` to the localPlugins list in your `~/.hyper.js` config file.

    localPlugins: [
    	'hyper-omni'
    ]

#### Activating theme

1.  Start Hyper
2.  Go to `View -> Full Reload` or pressing `Cmd + Shft + R`
